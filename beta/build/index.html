Rules

Hover On

  • The image switches from grayscale to full color.

  • The image also zooms in slightly

  • The card background color changes to gradient

  • cursor changes to pointer

Hover Off

  • The image switches from full color to grayscale.

  • The image also zooms out slightly to default.

  • The card background color changes from the gradient to default.

  • cursor changes back to arrow

  • menu bar slides to close
